ArcGIS Online Items
Any item in AGOL that could be thought of as a spatial layer/table/image. Ie. everything OTHER than a Webmap or Webmapping.
Groups
A group is created with specific permissions to allow members of the group to access only the tools and data relating to their work and therefore not be able to touch anyone elses.
Layer Sources
The source layers that are rendered in an MXD in ArcMap. One per mxd only.
MXDs
An ArcMap session file. It is where the authoring of the web service happens.
Webmaps
A map built in AGOL that contains one or many layers (ArcGIS Items).
Webmap Apps
A prettier fancy pants app that sources a Webmap and adds additional functionality like printing and heaps of other cool stuff.
Web Adapters
Allows the creation of a web alias that points to the correct server (which may change over time, so by using a web alias the connections to layers can be permanent).
Web Services
Created in ArcGIS Server (within DPaW) these web services are consumed in AGOL and become “layers” or “AGOL Items”.
AGOL Users
These are AGOL “named users”. DPaW has a limit of 154. Don’t hand them out willy nilly.
